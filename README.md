# GenshinImpact_Start
遇见白屏 -> 原神，启动！  C#、Winform、GenshinImpact。   
   
通过注册表，自动找到原神安装路径。   
通过注册表，禁用启动原神时的UAC弹窗，扫清启动障碍！   
   
当前软件仅支持Windows运行，且需要安装.Net运行支持。
